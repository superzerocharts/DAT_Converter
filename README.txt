DAT Converter

DAT Converter is a small portable converter for compatible .dat video files exported by supported recording software.

How to use:
1. Run DatConverter.exe.
2. Click Add Files to add one or more .dat files, or drag .dat files onto the app window.
3. Click Add Folder to scan a folder for .dat files.
4. Choose MP4 or MKV.
5. Choose Fast mode first. Use Full mode only if Fast output has playback issues.
6. Choose the Source FPS option that matches the recording. Use Auto Detect when available.
7. Choose Same folder as source file, or choose an output folder.
8. Click Start Queue.
9. Use Show Details only if you need technical conversion information.

Queue behavior:
- The queue processes one file at a time.
- You can add more .dat files while the queue is running.
- You can drag .dat files onto the app while the queue is running.
- Folders cannot be dropped while the queue is running. Use Add Folder before starting the queue if you need to scan a folder.
- Stop After Current lets the current file finish, then stops before the next file.
- Cancel Current cancels the current conversion and stops the queue.

Notes:
- DAT Converter only supports compatible .dat video files from supported recording exports.
- It is not a general video converter.
- It does not modify, overwrite, rename, or delete source .dat files.
- Existing output files are not overwritten.
- If a matching output file already exists, the queue row is marked Exists.
- Double-click a Ready or Exists row to change the save location or filename.
- Non-video or unsupported .dat files are skipped or marked unsupported.
